# PyWork: Applied Machine Learning & Synthetic Data 

This repository contains a collection of Jupyter Notebooks and Python scripts focused on applied machine learning, exploratory data analysis (EDA), and synthetic data generation. 

## Key Areas of Focus:
* **Predictive Modeling:** End-to-end regression and classification pipelines for predicting outcomes in various domains (e.g., Electric Vehicle [EV] metrics, Housing Prices).
* **Synthetic Data Generation (SDV):** Hands-on implementations using the Synthetic Data Vault (SDV) to generate realistic, privacy-preserving synthetic data for customer churn, real bank data, and housing records.
* **Exploratory Data Analysis:** Comprehensive EDA workflows to clean, visualize, and prepare data for model training.

## Featured Notebooks:
* `ML_Hands_On.ipynb` & `EDA_ModelPrediction.ipynb`: Foundational ML and EDA concepts.
* `HousingPrediction.ipynb`: Predicting real estate prices using regression techniques.
* `EVRegression.ipynb` & `EVLogisticRegression.ipynb`: Predicting Electric Vehicle (EV) characteristics.
* `Hands_on_Generating_Synthetic_House_Data_with_SDV.ipynb`: Deep dive into using SDV for data augmentation.
* `custChurn_syn_Data.ipynb` & `SDVRealBankData.ipynb`: Applying synthetic data models to banking and customer churn datasets.
